* pkg-config


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/taylorlcling/AreYouThere.git
   ```
2. Generate Build Files using cmake
   ```sh
   cmake .
   ```
3. Build Program
   ```sh
   make
   ```
4. Run Resulting Executable
   ```sh
   ./AreYouThere
   ```
5. Press any key to Exit
